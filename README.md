# 💄 Glam Store - متجر المكياج

## 📁 هيكل المشروع

```
makeup-store/
├── server.js          ← الخادم (Backend)
├── database.json      ← قاعدة البيانات
├── package.json
└── public/
    └── index.html     ← الواجهة (Frontend + JS)
```

---

## 🚀 كيف تشغل المشروع

### الخطوة 1 - تثبيت Node.js
حمّل Node.js من: https://nodejs.org

### الخطوة 2 - فتح Terminal في مجلد المشروع
```bash
cd makeup-store
```

### الخطوة 3 - تثبيت المكتبات
```bash
npm install
```

### الخطوة 4 - تشغيل السيرفر
```bash
node server.js
```

### الخطوة 5 - فتح المتجر
افتح المتصفح على: **http://localhost:3000**

---

## ✅ الميزات

- 🛍️ **عرض المنتجات** - يجيب المنتجات من قاعدة البيانات
- ➕ **إضافة منتج** - فورم لإضافة منتجات جديدة
- 🗑️ **حذف منتج** - حذف أي منتج
- 🛒 **إضافة للسلة** - إضافة منتجات للسلة
- ❌ **إزالة من السلة** - إزالة منتجات من السلة
- 💰 **حساب المجموع** - يحسب المجموع الكلي تلقائياً

---

## 🔗 الـ API Endpoints

| Method | URL | وظيفته |
|--------|-----|---------|
| GET | /api/products | جلب كل المنتجات |
| POST | /api/products | إضافة منتج |
| DELETE | /api/products/:id | حذف منتج |
| GET | /api/cart | جلب السلة |
| POST | /api/cart | إضافة للسلة |
| DELETE | /api/cart/:id | إزالة من السلة |

---

## 🗄️ قاعدة البيانات

ملف `database.json` هو قاعدة البيانات. كل شي يتحفظ فيه تلقائياً.
